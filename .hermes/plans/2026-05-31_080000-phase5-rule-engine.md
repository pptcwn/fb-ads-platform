# Phase 5: Rule Engine

> **Goal:** ให้用户可以ตั้งกฎ auto เช่น "ถ้า CPA > 200 → หยุด campaign" หรือ "ถ้า CTR < 1% → ลด budget" โดยระบบจะ eval กฎทุกๆ 5 นาที

**Architecture:**
- NestJS Rules module (CRUD + Engine + Scheduler)
- Prisma models Rule/RuleLog มีอยู่แล้วใน schema ✅
- Cron job ด้วย `@nestjs/schedule` ทุก 5 นาที
- Frontend Rules page สำหรับสร้าง/ดู/เปิดปิดกฎ
